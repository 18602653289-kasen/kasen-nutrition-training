## 每个专题的标准结构

```
专题编号-名称/
├── README.md              ← 学习路线图
├── 学前测试.md             ← 课前摸底
├── 知识点手册.md           ← 深度教材（含图片）
├── 音频/                   ← 播客音频
├── PPT/                   ← 课程课件
├── 逐字稿/                 ← 逐字文字稿
├── 大纲/                  ← 课程框架
├── 考试/                  ← 综合测试+答案
├── 学习卡/                ← 速记卡片
└── 关联资源.md            ← 知识库交叉链接
```

---

## 使用建议

1. **新营养师入职** → 按专题顺序学习 + 完成考试
2. **老员工复习** → 先做学前测试 → 只学薄弱的专题
3. **日常查阅** → 直接搜索知识点手册或看学习卡
4. **知识库联动** → 每个专题与专业知识、业务实战、标准法规板块互相引用

## 关于音频伴学

本系列培训课程已引入播客式音频伴学：
- 每个专题按模块拆分为 2-3 段音频
- 原料来自精炼后的逐字稿
- 生成后嵌入知识点手册，可边听边看
