# 评分规则

> 依据《建美元素·专家助理培训考核制度》制定

## 一、三级考核通过标准

| 层级 | 通过（≥） | 补考区间 | 淘汰（＜） | 补考次数 | 补考通过线 |
|------|-----------|---------|-----------|---------|-----------|
| 📅 **每日考核** | 60分 | 40 ≤ X ＜ 60 | 40分 | 1次 | ≥ 60分 |
| 📆 **每周考核** | 70分 | 50 ≤ X ＜ 70 | 50分 | 1次 | ≥ 70分 |
| 📅 **每月考核** | 80分 | 60 ≤ X ＜ 80 | 60分 | 1次 | ≥ 80分 |

## 二、考试构成

### 每日考核（30-40分钟）
- 满分：100分
- 题型：选择题（40分）+ 判断题（20分）+ 简答题（40分）
- 当日考核，当日出结果
- 补考：1次机会，须在成绩公布后1个工作日内完成

### 每周考核（60分钟）
- 满分：100分
- 题型：选择题（30分）+ 判断题（20分）+ 简答题（30分）+ 案例分析题（20分）
- 每周五下午进行
- 补考：1次机会

### 每月考核（90-120分钟）
- 满分：100分
- 题型：选择题（20分）+ 简答题（30分）+ 案例分析题（30分）+ 方案设计题（20分）
- 月末进行
- 补考：1次机会

## 三、主观题关键词评分规则

主观题（简答题、案例分析、方案设计）采取 **关键词匹配+人工复核** 的评分方式：

### 评分步骤
1. 每道题预设核心关键词（3-5个）
2. 学员答案中每命中一个关键词得对应分数
3. 关键词覆盖该题主要得分点
4. 阅卷人可根据答案完整性进行 ±20% 的分数调整

### 关键词设计原则
- 每个关键词权重均等
- 关键词必须是题干中明确要求的知识点
- 允许同义词、近义词匹配
- 专业术语必须精确匹配

### 示例
```
题目：请写出胃酸的三大任务
关键词：①激活胃蛋白酶原 ②杀菌 ③发信号
评分：命中1个→5分，命中2个→3.3分，命中3个→5分（满分）
调整：表述完整无歧义→满分，缺细节→扣01分
```

## 四、结果判定流程

```
总成绩（客观+主观）
      │
  ┌───┴───┐
  │       │
  ▼       ▼
≥通过线  ＜通过线
  │       │
  │   ┌───┴───┐
  │   │       │
  │   ▼       ▼
  │ ≥补考线  ＜补考线
  │   │       │
  │   ▼       ▼
  │  补考    淘汰
  │   │
  │   ▼
  │ ≥补考通过线 → 通过
  │ ＜补考通过线 → 淘汰
  │
  ▼
 通过
```

## 五、红线行为（直接淘汰）
- 考核作弊（任何形式）
- 触碰服务红线（承诺"治愈""停药"等违规话术）
- 旷课半天及以上
- 累计迟到/早退超过3次
