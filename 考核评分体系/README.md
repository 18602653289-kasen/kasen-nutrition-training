# 🎯 考核评分体系

> 基于建美元素·专家助理培训考核制度，构建统一的线上评分与判卷系统

---

## 体系架构

```
考核评分体系/
├── README.md                     ← 本文件（体系说明）
├── 评分规则.md                   ← 三级考核评分规则的完整说明
├── 判卷后台.html                ← 独立判卷页面（核心工具）
├── 每日考核记录.xlsx             ← 考核成绩记录模板
├── 案例库/                       ← 综合案例评分标准库
│   ├── README.md
│   ├── 消化系统案例评分标准.md
│   ├── 肝病案例评分标准.md
│   └── ... （每专题一个）
└── 更新日志.md
```

## 三级考核标准

| 层级 | 通过（≥） | 补考区间 | 淘汰（＜） | 补考次数 | 补考通过线 |
|------|-----------|---------|-----------|---------|-----------|
| 📅 **每日考核** | 60分 | 40 ≤ X ＜ 60 | 40分 | 1次 | ≥ 60分 |
| 📆 **每周考核** | 70分 | 50 ≤ X ＜ 70 | 50分 | 1次 | ≥ 70分 |
| 📅 **每月考核** | 80分 | 60 ≤ X ＜ 80 | 60分 | 1次 | ≥ 80分 |

## 判卷流程

```
学员作答 ─→ 提交答案
              │
      ┌───────┴───────┐
      ▼               ▼
  客观题             主观题
（单选/填空）       （简答/案例）
      │               │
      ▼               ▼
 自动判卷          AI辅助判卷
（对错判定）      （关键词匹配评分）
      │               │
      └───────┬───────┘
              ▼
        综合评分
              │
         ┌────┴────┐
         ▼         ▼
     通过/补考    淘汰
```

## 判卷后台功能

独立 HTML 页面（`判卷后台.html`），无需部署，浏览器直接打开即可使用：

| 功能 | 说明 |
|------|------|
| **考试配置** | 选择考试类型（日考/周考/月考）、设置及格线 |
| **学员管理** | 录入姓名、考场信息 |
| **客观题判卷** | 选择题自动匹配答案，填空关键词匹配 |
| **主观题判卷** | 输入关键词 → 自动高亮匹配 → 辅助评分子项 |
| **评分结果** | 自动计算总分、判定结果（通过/补考/淘汰） |
| **成绩导出** | 生成成绩报告，可复制/打印 |
